## 🚀 Featured Projects

### 🏠 [RentSmart v2](https://github.com/yashsabne/rentsmart_v2) — Property Rental Marketplace
> *Monolith → Microservices. Built for scale, deployed with Docker.*

A production-ready rental platform rebuilt from scratch as **6 independently deployable microservices**. Each service owns its own database, logic, and deployment — the way real-world distributed systems work.

| Service | What it does |
|---|---|
| 🔐 Auth Service | JWT login, email verification, rate limiting (Redis), Google + Microsoft OAuth |
| 🏘 Property Service | CRUD listings, Cloudinary image uploads (8 photos/listing), smart filters + recommendations |
| 💳 Payment Service | Razorpay order creation + HMAC-SHA256 signature verification, access gating, listing promotion |
| 💬 Chat Service | Real-time Socket.IO messaging, delivery tracking (sent → delivered → read), online presence |
| 🔔 Notification Service | Decoupled transactional email via Nodemailer |
| 📊 Activity Service | Centralised user event logging for analytics and audit trails |

**Stack:** `React 19` `Vite 8` `Node.js` `Express 5` `MongoDB` `Redis` `Socket.IO` `Razorpay` `Cloudinary` `Docker` `Passport.js` `JWT` `Tailwind CSS v4`
